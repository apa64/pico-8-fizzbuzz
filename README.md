# Naive FizzBuzz for PICO-8

> Extremely simplistic FizzBuzz solution for PICO-8.

## Getting Started

### Prerequisites

- [PICO-8](https://www.lexaloffle.com/pico-8.php)

### Installation

Start up PICO-8. Load and run the `.p8` cartridge.

```pico-8
load fizzbuzz-naive-elseif.p8
run
```
